# Modular I/O Module

Development area for Maker Pi USB coprocessor experiments (sensors, buttons, OLED).

## Goals

- USB serial link and heartbeat
- MicroPython/CircuitPython workflow validation
- ToF/ultrasonic and control-input experiments
- ROS2 bridge-ready data protocol

## Starter Scripts

- `src/io_smoke.py`
- `src/io_experimental.py`
