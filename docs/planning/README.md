# Planning Hub

This folder centralizes planning artifacts that were previously split between Cursor plan storage and repository docs.

## Canonical Files

- Active architecture plan: `docs/ball-rotation-architecture.md`
- Active project backlog/TODO: `docs/project-todo.md`
- Active conventions: `docs/conventions.md`

## Archived/Imported Planning Artifacts

- Imported Cursor plan snapshot:
  - `docs/planning/sensor-platform-execution-plan-2026-04-22.md`
- Project TODO snapshot at migration time:
  - `docs/planning/project-todo-snapshot-2026-04-23.md`

## Usage Rule

- Keep day-to-day updates in canonical files (`architecture`, `project-todo`, `conventions`).
- Store milestone snapshots and externally generated planning exports in this folder.
- Treat files in this folder as historical references, not the primary editable planning source.
