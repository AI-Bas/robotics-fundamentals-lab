# Vision Camera Module

Development area for Raspberry Pi global shutter camera + AI HAT+ experiments.

## Goals

- local and SSH-accessible video feed
- low-rate browser/HMI preview stream
- OpenCV-based ball and seam feature tracking
- ROS2-ready outputs for control and HMI
